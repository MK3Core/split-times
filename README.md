Track F1, IMSA, WEC, and WRC races in one place. Download personalized .ics calendars with timezone support.

**Built with Vite + React**

## Features

- **4 Racing Series** - F1, IMSA, WEC, WRC (2026 schedules)
- **Timezone Support** - 24 timezones, auto-converts all race times
- **Light/Dark Theme** - Terminal-inspired dark mode
- **Download Calendars** - .ics files for Apple Calendar, Google Calendar, Outlook
- **Responsive** - Works on desktop, tablet, mobile

## Quick Start

```bash
npm install
npm run dev
```

Build for production:
```bash
npm run build
```
